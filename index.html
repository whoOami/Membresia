Membresia

"Membresia" es un software capaz de adaptarse a cualquier entidad que requiera gestionar por grupos sus miembros.

Bienvenido a este proyecto.

Membresia permite un manejo ágil y sencillo de cada uno de los integrantes de una entidad y asociarlos a uno o varios grupos, tiene las transacciones básicas de un CRUD (crear, visualizar, modificar y eliminar) para los grupos e integrantes, y por consiguiente el movimiento de integrantes entre cada uno de los diferentes grupos.

Caracteristicas

Esto es una iniciativa independiente que aspira ser lo más minimalista posible sin decantar funcionalidades y transacciones, para el backend se contruyó una API REST FULL sobre Laravel Framework 5.1 y para el frontend se tomó como base una plantilla AngularJS, y ya que tomé este camino, y el JSON WEB TOKEN parecia interesante, decidi implementar un JWT AUTHENTICATION y fué gracias al Satellizer de sahat (graduado de la antes llamada Hacker School que según palabras de sus propios bloggers cambió su nombre a Recurse Center porque Hacker suena a malo y Escuela suena a peor :p), el jwt-auth de tymondesigns, y obviamente a toda la ducumentación que circula la red, (enserio muchachos sin ustedes no lo hubiese logrado :'))

Instalación (en la raíz del proyecto)

  • chmod 777 -R vendor/
  • chmod 777 -R storage/
  • cp .env.example .env
  • dentro del .env se configuran los datos de conexión a la DB.
  • php artisan migrate --seed
  • chmod 777 -R boot/
